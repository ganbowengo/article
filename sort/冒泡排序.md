#### 冒泡排序（Bubble Sort）

**基本思想**：两个数比较大小，较大的数下沉，较小的数冒起来。

**过程**：
- 比较两相邻的数的大小，大与小的调换位置，大数在后面

**复杂度**：
- 时间复杂度O(n^2)
- 空间复杂度O(1)

```
    // 简陋版 较小数上浮 以下是第一层的起点位置小数上浮时，最前面的数一定最小
    /*
        * * * * *
          * * * *
            * * *
              * *
                *
    */
    function bubbleSort(arr){
        let len = arr.length
        let i = 0
        let j = 0
        while(i < len){
            j = len
            while(j-- > i){
                if(arr[j] < arr[j-1]){
                    let temp = arr[j]
                    arr[j] = arr[j-1]
                    arr[j-1] = temp
                }
            }
            i++
        }
    }
    
    // 简陋版 较大数下沉 以下是第二层的终点位置大数下沉时，最后面的数一定最大
    /*
        * * * * *
        * * * *
        * * *
        * *
        *
    */
    function bubbleSort(arr){
        let len = arr.length
        let i = 0
        let j = len
        while(i < len){
            j = 0
            flag = false
            while(j < len - i){
                if(arr[j] < arr[j-1]){
                    let temp = arr[j]
                    arr[j] = arr[j-1]
                    arr[j-1] = temp
                }
                j++
            }
            i++
        }
    }
```

**冒泡算法的改进**：
如果在第二层for循环中，如果不发生交换，则代表数据已经排好序了，不需要继续排序。利用falg作为标志记录第二层是否交换过，没有交换时，代表排序已完成

```
    // 加强版 较小数上浮
    function bubbleSort1(arr){
        let len = arr.length
        let i = 0
        let j = 0
        let flag = false
        while(i < len){
            j = len
            flag = false
            while(j-- > i){
                if(arr[j] < arr[j-1]){
                    let temp = arr[j]
                    arr[j] = arr[j-1]
                    arr[j-1] = temp
                    flag = true
                }
            }
            if(!flag) break
            i++
        }
    }

```
